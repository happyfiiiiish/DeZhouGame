# 双人局域网德州扑克

一个最小可玩的双人德州扑克网页游戏，支持同一局域网下两位玩家对战。

## 当前功能

- 房主创建房间，另一位玩家输入房间码加入
- 双方初始各有 `50` 筹码
- 双人盲注轮换：小盲 `1`，大盲 `2`
- 每位玩家各发 `2` 张手牌，桌面最多 `5` 张公共牌
- 每一街都支持下注行动：
  - `加注`：短按默认 `+1`
  - `加注`：长按后可输入自定义数字
  - `跟随`
  - `弃牌`
- 只有当前这一街双方下注平齐后，房主才可以继续点击 `开牌`
- `river` 结束后点击 `开牌` 会直接比牌，并显示双方最大牌型
- 弃牌时，未弃牌方直接赢得当前底池
- 平局时，底池默认保留到下一手，交给下一位真正的赢家
- 如果平局且会导致一方或双方筹码变成 `0`，则该手改为当场平分底池
- 一方筹码归 `0` 后，整场比赛结束；房主可点击 `重新开赛`

## 本地启动

如果终端里已经能直接使用 `node` 和 `npm`：

```bash
npm install
npm run dev
```

如果当前 Windows 终端没有把 Node 加进 `PATH`，可以直接用下面的命令：

```powershell
$env:Path = 'C:\Program Files\nodejs;' + $env:Path
& 'C:\Program Files\nodejs\npm.cmd' install
& 'C:\Program Files\nodejs\npm.cmd' run dev
```

默认端口：

- 前端：`5173`
- 后端：`3001`

## 局域网游玩

1. 在作为主机的电脑上启动项目
2. 查询主机电脑在局域网中的 IPv4 地址
3. 让两位玩家都在浏览器中访问 `http://主机IP:5173`
4. 一位玩家创建房间，另一位玩家输入房间码加入

客户端默认会连接到当前页面同主机的 `3001` 端口。

如果需要手动指定后端地址，可以在前端启动前设置：

```bash
VITE_SERVER_URL=http://你的后端IP:3001
```

## 规则说明

- 本项目是双人版本，采用 heads-up 的基本流程
- 庄家同时也是小盲
- `preflop` 由小盲先行动
- `flop / turn / river` 由大盲先行动
- 当前版本不实现边池
- 加注上限按有效筹码限制，避免出现对手无法匹配的超额下注

## 当前范围

- 只支持双人
- 已支持筹码、盲注、加注、跟随、弃牌、平局延续底池和比赛重开
- 暂不支持边池、旁观、聊天、音效、计分榜
- 玩家断线后需要重新加入房间
