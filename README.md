# 双人局域网德州扑克

一个最小可玩的双人德州扑克网页游戏：

- 房主创建房间，另一位玩家输入房间码加入
- 双方各发 2 张手牌
- 第 1 次开牌翻 3 张公共牌，第 2 次翻 1 张，第 3 次翻 1 张
- 第 4 次点击“开牌”直接比牌并显示双方最大牌型
- 房主可以在任意阶段直接点击“新一局”，立即重置当前局并重新发牌

## 本地启动

如果你的终端里 `node` / `npm` 已经可用：

```bash
npm install
npm run dev
```

如果当前 Windows 终端没有把 Node 加进 `PATH`，可以直接用绝对路径：

```powershell
$env:Path = 'C:\Program Files\nodejs;' + $env:Path
& 'C:\Program Files\nodejs\npm.cmd' install
& 'C:\Program Files\nodejs\npm.cmd' run dev
```

默认端口：

- 前端：`5173`
- 后端：`3001`

## 局域网游玩

1. 在作为主机的电脑上启动前后端。
2. 查询主机电脑在局域网中的 IP 地址。
3. 让两位玩家都在浏览器中访问 `http://主机IP:5173`。
4. 一位玩家创建房间，另一位玩家输入房间码加入。

客户端默认会连接到当前页面同主机的 `3001` 端口；如果需要手动指定服务端地址，可以在前端启动前设置 `VITE_SERVER_URL`。

## 当前范围

- 只支持双人
- 没有筹码、下注、弃牌、计分和旁观
- 玩家断线后需要重新加入房间
