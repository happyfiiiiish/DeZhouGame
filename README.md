# 多人局域网德州扑克

一个可在同一局域网内游玩的网页德州扑克项目。当前版本支持 `2-8` 人同桌、房主开赛后锁房、多人下注轮、主池/边池结算、历史牌局记录、筹码动画，以及更适合一屏操作的紧凑牌桌布局。

## 当前功能

- 房主创建房间，其他玩家在大厅阶段输入房间码加入。
- 每位玩家初始 `50` 筹码。
- 支持 `2-8` 人同桌。
- 房主点击开始整场后，房间立即锁定，不允许新玩家中途加入。
- 固定盲注：
  - 小盲 `1`
  - 大盲 `2`
- 使用标准多人德州流程：
  - 每位仍有筹码的玩家发 `2` 张手牌
  - 桌面最多 `5` 张公共牌
  - `preflop / flop / turn / river` 每一街都要先完成下注，房主才可以继续开牌
- 支持下注操作：
  - `加注`：直接在输入框中输入“这一街总共加到多少”
  - `过牌 / 跟随`
  - `弃牌`
- 支持主池与边池结算。
- 摊牌后显示每位仍在手中的玩家可组成的最佳牌型。
- 支持多人淘汰赛：
  - 筹码归 `0` 的玩家会留在桌上旁观
  - 直到只剩 `1` 位玩家仍有筹码时，整场比赛结束
  - 房主可点击 `重新开赛`，把桌上玩家筹码统一重置为 `50`
- 支持轻量筹码动画：
  - 盲注、跟注、加注时，筹码从座位飞向底池
  - 弃牌收池、摊牌收池、分池时，筹码从底池飞向赢家
- 牌桌 UI 已优化为紧凑布局：
  - 非房主看不到房主操作面板
  - 玩家列表显示每位玩家当前筹码
  - 每位玩家面前显示“这一整手累计投入”的筹码托盘
  - 中央牌桌、顶部状态和底部操作区尽量减少互相遮挡

## 历史牌局记录

- 牌桌右下角有 `历史记录` 入口。
- 可查看本场比赛从开始到当前为止的所有已结束手牌。
- 每条记录先显示一行概要，例如：
  - `玩家 A 两对胜 玩家 B 一对，收下 12 筹码`
  - `玩家 C 弃牌，玩家 D 收下 8 筹码`
  - `玩家 A / 玩家 C 完成分池，总计 24 筹码`
- 点开条目后可以查看：
  - 公共牌
  - 当前查看者有权限看到的手牌
  - 最佳牌型与最佳五张
  - 主池 / 边池分配
  - 每位玩家本手净输赢

## 信息可见性规则

- 自己始终可以看到自己的手牌。
- 已进入正常摊牌的玩家，公开手牌可被其他人看到。
- 弃牌且未摊牌的玩家，其手牌默认对其他人隐藏。
- 弃牌玩家本人在历史记录中仍可看到自己那一手的牌。
- 淘汰旁观者也不能通过主视角或历史记录看到别人弃掉但未公开的手牌。
- 历史记录展示的信息，与主视角当时本应知道的信息保持一致。

## 规则说明

- 房主第一次点击 `开始整场` 后，房间立刻锁定。
- 已淘汰玩家不再参与发牌和行动，但会保留在桌面中显示。
- 只有当前行动玩家才能下注或弃牌。
- 只有当前这一街下注平齐后，房主的 `开牌` 按钮才会可用。
- `river` 阶段下注结束后，房主点击 `开牌` 会触发摊牌与结算。
- 多人版本采用标准德州结算：
  - 不再使用旧双人版本的“平局底池留到下一手”
  - 所有主池和边池都会在当前手牌结束时立即分配完成

## 本地启动

如果终端里已经可以直接使用 `node` 和 `npm`：

```bash
npm install
npm run dev
```

如果当前 Windows 终端没有把 Node 加进 `PATH`，可以使用：

```powershell
$env:Path = 'C:\Program Files\nodejs;' + $env:Path
& 'C:\Program Files\nodejs\npm.cmd' install
& 'C:\Program Files\nodejs\npm.cmd' run dev
```

默认端口：

- 前端：`5173`
- 后端：`3001`

## 局域网游玩

1. 在作为主机的电脑上启动项目
2. 确认其他手机或电脑与主机处于同一局域网
3. 找到主机电脑的 IPv4 地址
4. 所有设备在浏览器中访问：

```text
http://主机IP:5173
```

例如：

```text
http://192.168.1.119:5173
```

客户端默认会自动连接同一主机的 `3001` 端口。

如果需要手动指定后端地址，可在前端启动前设置：

```bash
VITE_SERVER_URL=http://你的后端IP:3001
```

## 开发命令

- 启动前后端：

```bash
npm run dev
```

- 仅启动前端：

```bash
npm run dev:client
```

- 仅启动后端：

```bash
npm run dev:server
```

- 构建前端：

```bash
npm run build
```

- 运行测试：

```bash
npm test
```

## 当前范围

- 已支持 `2-8` 人同桌、边池、淘汰赛、直接输入加注、紧凑牌桌布局、筹码动画、历史记录和可见性一致性规则。
- 暂不支持下注计时、聊天、旁观者中途加入、断线重连、筹码滑杆和音效。
